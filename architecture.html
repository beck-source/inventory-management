Frontend Architecture

Composables (Shared State)

useFilters.js

  • Singleton reactive filter state
  • selectedPeriod, selectedLocation, selectedCategory, selectedStatus
  • getCurrentFilters() maps UI state to API params
  • hasActiveFilters computed for reset button

useAuth.js

  • Mock current user with name, role, department
  • 4 pending tasks with priority and due dates
  • getInitials(), logout()

useI18n.js

  • English / Japanese locale toggle
  • Currency mapping: en→USD, ja→JPY
  • translateProductName(), translateWarehouse()
  • Placeholder replacement in translation strings
