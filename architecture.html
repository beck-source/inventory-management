Request Data Flow
01
User Interaction
User selects filters (period, warehouse, category, status) in FilterBar
02
Reactive State
useFilters() composable updates global reactive refs; watchers in views trigger re-fetch
03
API Call
api.js builds URL with query params via Axios: ?warehouse=Tokyo&month=2025-03
04
Filter & Aggregate
FastAPI chains apply_filters() + filter_by_month() on in-memory lists, then computes KPIs
05
Validation
Pydantic serialises filtered results to typed response models
06
Render
Vue component receives JSON, updates refs; computed properties derive display data

Query Params
GET /api/…?filters
Each filter defaults to all (skip). The backend function apply_filters(items, warehouse, category, status) chains only non-all conditions. Quarter values (Q1-2025) are expanded to month lists before matching. Inventory endpoints ignore the month dimension (no time axis on stock data).
