→
02

→
03
Axios request
Non-'all' filters serialized into URL query params
→
04
FastAPI filters
apply_filters() and filter_by_month() narrow in-memory arrays
→
05

→
06
Vue renders
Computed properties derive UI state from raw response refs

Orders

Order status breakdown — Delivered, Shipped, Processing, Backordered; expandable rows
Demand

Revenue vs costs, monthly cost flow, transactions list

Quarterly performance and month-over-month trends with bar charts

Inventory shortages by priority (high/medium/low), days delayed, quantity gaps

API Endpoints

GET /api/dashboard/summary KPI aggregates for dashboard cards warehouse · category · status · month
GET /api/inventory All inventory items warehouse · category
GET /api/inventory/{id} Single inventory item
GET /api/orders All orders warehouse · category · status · month
GET /api/orders/{id} Single order
GET /api/demand Demand forecasts
GET /api/backlog Backlog/shortage items with purchase order flag
GET /api/spending/summary Procurement, operational, labor, overhead totals
GET /api/spending/monthly 12-month cost breakdown by category
GET /api/spending/categories Spending grouped by cost category
GET /api/spending/transactions Recent financial transactions
GET /api/reports/quarterly Q1–Q4 2025 stats: orders, revenue, fulfillment rate
GET /api/reports/monthly-trends Month-over-month order count, revenue, delivered

Data Files — server/data/

inventory.json
SKUs, quantities, reorder points, unit costs, warehouse locations
orders.json
Customer orders with items[], status, dates, total value
demand_forecasts.json
Current vs forecasted demand, trend direction per SKU
backlog_items.json
Shortage records — quantity gaps, days delayed, priority
spending.json
Summary totals, monthly breakdown, and category spending
transactions.json
Financial transactions with vendor, amount, category
purchase_orders.json
Purchase orders linked to backlog items (currently empty)
