Composables (client/src/composables/)

  • useFilters.jsSingleton filter state shared across all views (warehouse, category, status, period)
  • useI18n.jsEN/JP translations, locale in localStorage, currency follows locale (USD/JPY)
  • useAuth.jsMock auth state, user info, initials, task list

Key Design Choices

  • StateNo Pinia/Vuex — module-level refs in useFilters.js act as singleton
  • ChartsCustom SVG only, no third-party charting library
  • RoutingVue Router defined in main.js, 6 routes
