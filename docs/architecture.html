System Architecture

Factory Inventory Management System — Vue 3 + FastAPI + In-memory JSON

Overview

Frontend

Vue 3 SPA served by Vite dev server on port 3000

Vue 3 Vue Router Vite Axios

Backend

Python FastAPI REST API served by Uvicorn on port 8001

FastAPI Pydantic Uvicorn uv

Data Layer

JSON files loaded into memory at startup — no database

JSON Files In-memory

Testing

pytest + FastAPI TestClient for backend API tests

pytest httpx pytest-cov

Filter Data Flow

01 — USER
Filter Selection
FilterBar.vue updates refs in useFilters composable
→
02 — COMPOSABLE
getCurrentFilters()
Returns {warehouse, category, status, month} object
→
03 — API CLIENT
api.js
Builds URLSearchParams, skipping 'all' values. Axios GET request
→
04 — BACKEND
FastAPI Endpoint
Parses query params, calls apply_filters() + filter_by_month()
→
05 — RESPONSE
Pydantic + JSON
Validated model list serialized to JSON response
→
06 — FRONTEND
Reactive Update
ref updated, computed properties recalculate, Vue re-renders

API Endpoints

Method Path Filters Supported Returns
GET /api/inventory warehouse, category List[InventoryItem]
GET /api/inventory/{id} — InventoryItem
GET /api/orders warehouse, category, status, month List[Order]
GET /api/orders/{id} — Order
GET /api/dashboard/summary warehouse, category, status, month Summary stats object
GET /api/demand — List[DemandForecast]
GET /api/backlog — List[BacklogItem]
GET /api/spending/summary — Spending summary object
GET /api/spending/monthly — Monthly cost breakdown
GET /api/spending/categories — Category spending
GET /api/spending/transactions — Recent transactions
GET /api/reports/quarterly — Quarterly performance
GET /api/reports/monthly-trends — Month-over-month trends
POST /api/purchase-orders — PurchaseOrder (created)
GET /api/purchase-orders/{backlog_id} — PurchaseOrder

Data Files

inventory.json

Stock levels per warehouse

  • id, sku, name, category
  • warehouse, quantity_on_hand
  • reorder_point, unit_cost
  • Warehouses: SF, London, Tokyo
  • Categories: Circuit Boards, Sensors, Actuators, Controllers, Power Supplies
orders.json

Customer orders for Jan–Sep 2025

  • id, order_number, customer
  • items[], status, total_value
  • order_date (ISO 8601)
  • Statuses: Delivered, Shipped, Processing, Backordered
demand_forecasts.json

30-day demand predictions

  • item_sku, item_name
  • current_demand, forecasted_demand
  • trend: increasing / stable / decreasing
backlog_items.json

Short-stock / delayed order items

  • order_id, item_sku
  • quantity_needed, quantity_available
  • days_delayed, priority
  • Priorities: low / medium / high
spending.json

Cost breakdowns by type & month

  • spending_summary (totals + % change)
  • monthly_spending (12 months)
  • category_spending
  • Types: procurement, operational, labor, overhead
transactions.json

Individual financial transactions

  • id, date, description
  • category, warehouse, amount
  • vendor, type
  • Types: Purchase, Operational, Labor
