Tech Stack

Frontend
Vue 3 Client
Runs on localhost:3000
Vue 3 Composition API Vite Vue Router Axios
Backend
FastAPI Server
Runs on localhost:8001
Python 3.14 FastAPI Uvicorn Pydantic v2 uv
Data Layer
JSON Mock Data
In-memory at startup
inventory.json orders.json demand_forecasts.json backlog_items.json spending.json

Request Data Flow

User Interaction Filter bar / button click
→
Vue Component Reactive ref / computed
→
api.js Axios HTTP call
→
FastAPI Endpoint Filter + validate
→
mock_data.py In-memory lists
→
Pydantic Model Serialise & return JSON
→
Vue Template Render UI

Frontend Views

/
Dashboard
KPI summary cards, inventory value, pending orders, backlog count
/inventory
Inventory
Stock levels, reorder points, warehouse & category filters
/orders
Orders
All orders by status; submitted restocking orders with lead times
/spending
Spending
Monthly spend, category breakdown, recent transactions
/demand
Demand Forecast
Forecasted vs current demand per SKU, trend indicators
/reports
Reports
Quarterly performance, monthly trends
/restocking NEW
Restocking
Budget slider, demand-driven restock recommendations, place orders
