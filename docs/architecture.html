Inventory Management — Architecture

Full-stack demo: Vue 3 + FastAPI with in-memory JSON mock data.

System Overview

Frontend

Vue 3 + Vite

Port 3000 · client/
  • Composition API, no Pinia
  • 6 views (Dashboard, Inventory, Orders, Demand, Spending, Reports)
  • Shared filter state via useFilters composable
  • Axios client in client/src/api.js
Backend

Python FastAPI

Port 8001 · server/
  • Single-module API in server/main.py
  • Pydantic models for validation
  • In-memory filtering by warehouse, category, status, month
  • CORS open for the Vite dev server
Data

JSON Mock Files

server/data/*.json
  • Loaded once at startup by mock_data.py
  • No database, no persistence
  • 7 files: inventory, orders, demand, backlog, POs, spending, transactions
