API Endpoints
Method Endpoint Filters Returns
GET /api/inventory warehouse, category List of inventory items
GET /api/inventory/{id} — Single inventory item
GET /api/orders warehouse, category, status, month List of orders
GET /api/orders/{id} — Single order
GET /api/dashboard/summary warehouse, category, status, month KPI totals (value, low-stock, pending)
GET /api/demand — Demand forecasts by SKU & trend
GET /api/backlog — Backlog items with PO flag
GET /api/spending/summary — Procurement / ops / labor / overhead
GET /api/spending/monthly — Monthly cost breakdown
GET /api/spending/categories — Category-level cost totals
GET /api/spending/transactions — Recent transaction log
GET /api/reports/quarterly — Quarterly KPIs (computed from orders)
GET /api/reports/monthly-trends — Monthly order/revenue trends
