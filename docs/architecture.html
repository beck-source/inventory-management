Factory Inventory Management System

Full-stack demo: Vue 3 + Vite frontend, FastAPI backend, JSON-backed in-memory store.

Workshop demo — no database, no auth, CORS allow-all

Tech Stack what's in the box

Three layers, each independently runnable. The data layer is a set of JSON files loaded into Python memory at startup.

Frontend

client/
  • Vue 3 Composition API
  • vue-router 4 createWebHistory
  • axios HTTP client
  • Vite 5 dev server, port 3000

Backend

server/
  • FastAPI routes + CORS
  • Pydantic response validation
  • Uvicorn ASGI, port 8001
  • uv env + dependencies

Data

server/data/
  • inventory.json SKUs by warehouse
  • orders.json 12 months of orders
  • demand_forecasts.json trend data
  • backlog_items.json + purchase_orders.json
  • spending.json + transactions.json

System Architecture layered view

A request from the browser passes through Vue's router and view, into the centralized axios client, across to FastAPI, through the in-memory loader, and finally reaches the static JSON files on disk.

Data Flow filter change → re-render

The 4 filters in FilterBar.vue drive the entire app. When any filter changes, the affected views fire fresh API calls; the backend filters in memory and returns Pydantic-validated lists; Vue's computed properties derive display values from the new raw data.

Filter System 4 dimensions, one component

The global FilterBar.vue drives every filterable endpoint. Selection state lives in a shared composable; affected views watch it and re-fetch via api.js. Each filter maps to a single query-string param.

Time Period

Month / quarter — applies to orders, dashboard summary, spending trends.

Warehouse

Filters inventory, orders, dashboard summary by physical location.

Category

Circuit Boards, Sensors, Actuators, Controllers — applies to inventory, orders, dashboard.

Order Status

Delivered, Shipped, Processing, Backordered — applies to orders and dashboard.

Inventory has no time dimension, so the time-period filter is a no-op for /api/inventory. Demand and backlog endpoints ignore all four filters.
